Passwords are terrible. Here's what we use instead

Passkey.
Use an Apple Passkey, Windows Hello, TouchID, FaceID, security key, or other device-based authenticator to sign in quickly. (This only works on devices you've already registered)
Email Code.
Get a secret code sent to your email address. It expires after five minutes, can only be used once, and can only be used in this browser session. This is great for signing in for the first time or on a new device.
